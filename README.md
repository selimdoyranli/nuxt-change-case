## Setup

1. Add `nuxt-change-case` dependency to your project

```bash
yarn add nuxt-change-case # or npm install nuxt-change-case
```

2. Add `nuxt-change-case` to the `modules` section of `nuxt.config.js`

```js
{
  modules: [
    'nuxt-change-case'
  ]
}
```
